# 💰 Expense Tracker

A modern and responsive Expense Tracker web application built using HTML, CSS, and JavaScript.

## Features

* ➕ Add expenses with title and amount
* 🗑️ Delete expenses instantly
* 💾 Data persistence using Local Storage
* 📊 Automatic total expense calculation
* 😀 Dynamic emoji detection for expense categories
* 🔄 Real-time UI updates with DOM Manipulation
* 📱 Responsive design for all devices
* ✨ Modern Glassmorphism User Interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## Concepts Practiced

* DOM Manipulation
* Event Handling
* Arrays and Objects
* Array Methods (forEach, filter, reduce)
* Dynamic Element Creation
* Template Literals
* Local Storage
* CRUD Operations

## Future Improvements

* Expense Categories
* Expense Editing
* Monthly Analytics
* Charts and Visual Reports
* Dark/Light Theme Toggle

This project helped strengthen JavaScript fundamentals by implementing real-world CRUD operations and data persistence in a practical application.
